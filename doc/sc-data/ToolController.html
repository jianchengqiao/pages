5.1. 模拟推送客户

URL: https://sc-data.focus-test.cn/tool/client/push

Type: POST

Content-Type: application/json; charset=utf-8

Description: 模拟推送客户

Body-parameters:

ParameterTypeRequiredDescription

id

int64

false

客户id，修改、删除必填，新增不填

projectId

int64

true

项目id

receiveType

int32

true

获取类型，1新增客户，3修改客户，4删除客户
另外，数据中心会根据是否传入客户id，手机号是否重复，重新判定获取类型

brokerId

int64

false

指定归属顾问，不传不指定，数据中心自动分配

name

string

true

姓名 {40}

avatar

string

false

头像

telephone

string

true

手机号 {11}

gender

int32

false

性别，1男，2女 {1,2}

age

int32

false

年龄 {0,200}

job

string

false

职业，What's your job? {20}

livingPlace

string

false

现居住地 {40}

bornPlaceId

int32

false

户口所在地城市id

source

int32

true

获取来源，1智能话机，source=sourceAction/100

sourceAction

int32

true

获取方式，101客户来电录入，102顾问去电录入，103兼职去电录入

mark

int32

false

标记，1未标记，2有效客户，3无效客户，4需再跟进 {1,4}

process

int32

false

看房状态(进度)，1未看房，2已看房，3已认购，4已草签，5已网签 {1,5}

intent

int32

false

意向级别，1 A-强烈，2 B-较高，3 C-一般，4 D-较低 {1,4}

remark

string

false

记录备注 {100}

tagIds

array

false

tag标签，只传id (ActualType: Long)

createTime

string

true

获取时间

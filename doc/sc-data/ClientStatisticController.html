4.1. 首页统计(含图表)

URL: https://sc-data.focus-test.cn/broker/statistics/

Type: GET

Content-Type: application/x-www-form-urlencoded;charset=utf-8

Description: 聚合接口，包括今日工作、客户标记环形图、近7天柱状图

Query-parameters:

ParameterValueTypeRequiredDescription

int32

false

int32

false

barChartUnit

int32

false

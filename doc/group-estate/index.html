1. 调用说明

1.1. 后台接口调用说明

URL: https://group-estate.focus-dev.cn/doc

Type: GET

Content-Type: application/x-www-form-urlencoded;charset=utf-8

Description: 

后台接口调用说明

  • 所有接口都经过group-auth 服务路由，不能直接调 group-estate 服务
  • 所有非 GET 请求经过 group-auth 服务路由后，会带上 operatorId, operatorName 参数，开发无需关心操作人，只需要传 cookie 或 token
  • 所有 body 里需要传 groupId 的，都可以拼在 query 上，后端会自动处理
  • 路由规则：

    • http://group-estate.focus-test.cn/backstage/estates?groupId=xxx

    ==>

    http://group-auth.focus-test.cn/group/estate/backstage/estates?groupId=xxx


    Response-example:

    Doesn't return a value.

    Curl-example:

    curl -X GET -k -i https://group-estate.focus-dev.cn/doc
